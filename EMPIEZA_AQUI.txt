==============================================
  ⚠️ IMPORTANTE - CONFIGURAR IMÁGENES
==============================================

Para que las imágenes funcionen:

1. Ve a tu proyecto Supabase:
   https://supabase.com/dashboard/project/dwwzqwcoqlasgvpniwiu

2. Settings > API
   - Copia "anon public"
   - Copia "service_role" (¡secreto!)

3. Edita .env y agrega:
   NEXT_PUBLIC_SUPABASE_ANON_KEY="..."
   SUPABASE_SERVICE_ROLE_KEY="..."

4. Storage > Data_bucket > Policies
   - Activa acceso público para INSERT
   - Activa acceso público para SELECT

  🎯 CÓMO FUNCIONA
==============================================

1. CREAR SERMÓN
   - Título: "El amor de Dios"
   - Pastor: Tu nombre
   - Fecha: Hoy

2. TOMAR NOTAS
   - Escribe mensajes como en WhatsApp
   - Cada punto principal = 1 mensaje
   - Adjunta fotos de diapositivas
   - Todo se guarda automáticamente

3. REVISAR
   - Abre el sermón
   - Ve todos los mensajes en orden
   - Click en imágenes para ampliar
