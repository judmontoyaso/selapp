Tu aplicación PWA "selapp" — Devocionales y Notas
está lista para usar. Selapp ofrece una experiencia elegante, cálida y moderna para tus devocionales.

==============================================
  📖 LEE ESTO PRIMERO
==============================================

1. Abre y lee: RESUMEN.md
   → Visión general completa del proyecto

2. Luego lee: INICIO.md  
   → Guía paso a paso para empezar

3. Ejecuta (Windows):
   setup.bat
   
   O ejecuta (Mac/Linux):
   bash setup.sh
   
   → Script interactivo de configuración

==============================================
  🚀 INICIO RÁPIDO (Manual)
==============================================

1. Configurar variables de entorno:
   cp .env.example .env
   (Luego edita .env con tus datos)

2. Configurar base de datos:
   npx prisma migrate dev --name init
   npx prisma generate

3. Poblar con datos de ejemplo:
   npm run db:seed

4. Iniciar aplicación:
   npm run dev

5. Abrir en navegador:
   http://localhost:3000
